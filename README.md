# 📊 Business Sales Dashboard – Power BI

## 📌 Project Overview
This project focuses on building an **interactive Business Sales Dashboard** using **Power BI** to analyze e-commerce sales data.

The dashboard helps users monitor overall sales performance, identify top-performing products, analyze monthly sales trends, and understand category-wise revenue distribution.

This project was developed as part of a **Data Analytics learning project** to demonstrate skills in **data visualization, data analysis, and business intelligence.**

---

## 📁 Dataset

**Source:** Superstore Dataset (Kaggle)  
**File Format:** Excel (.xlsx)

### Dataset Includes:
- Order Date  
- Product Name  
- Category and Sub-Category  
- Sales, Profit, and Quantity  
- Region  
- Customer Segment  

---

## 🛠️ Tools & Technologies

- **Power BI Desktop**
- **Microsoft Excel**
- **DAX (Data Analysis Expressions)**

---

## 🔧 Data Preparation

Before building the dashboard, the dataset was cleaned and transformed by performing the following steps:

- Removed duplicate records
- Handled missing or blank values
- Converted fields to correct data types (Date, Numeric)
- Renamed columns for better readability
- Created calculated measures using **DAX**

---

## 📌 Key Metrics (DAX Measures)

The following Key Performance Indicators (KPIs) were created:

- **Total Sales**
- **Total Profit**
- **Total Quantity Sold**
- **Total Orders**

---

## 📈 Dashboard Features

The dashboard includes several visualizations to provide meaningful insights:

- **KPI Cards** for overall business performance
- **Monthly Sales Trend Analysis**
- **Top 10 Best-Selling Products**
- **Sales Distribution by Category**
- **Interactive Filters (Slicers)** for:
  - Date
  - Category
  - Region

---

## 🔍 Key Insights

Some important business insights identified from the dashboard:

- The **Technology category generates the highest sales revenue**
- Sales show a **consistent upward trend over time**
- Certain regions contribute **significantly higher revenue**
- The **top 10 products contribute a major portion of total sales**
